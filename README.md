## About Docker image
## supported env vals
- DB_ADDR
- DB_PORT
- DB_USER
- DB_PASSWORD
- DB_DATABASE

## not supported env vals
- DB_VENDOR (only postgresql available)
- KEYCLOAK_USER
- KEYCLOAK_PASSWORD
- KEYCLOAK_FRONTEND_URL
